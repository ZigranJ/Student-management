# Student management

Student management is a CLI program to manage excel files

## Installation
to install and run:
```bash
git clone https://github.com/ZigranJ/Student-management.git
cd Student-management
python main.py
```

## Commands
```
average - Calculate averages of all students
top - Find the top student
sort - Sort the students based on averages
add - Add a new student
save - save the file
exit - exit the program
```
